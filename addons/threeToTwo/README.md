## Quick Overview

ThreeToTwo is a powerful Godot Engine plugin designed for converting 3D models into 2D textures. It provides a comprehensive solution for game developers who need to create 2D sprites, textures, and animations from 3D resources.

### Key Features

- **3D Model Import**: Supports FBX, GLTF, GLB, OBJ formats
- **Real-time Preview**: Interactive 3D view with camera controls
- **Material Switching**: Toggle between regular and normal materials
- **Animation Support**: Play and export 3D animations as 2D sequences
- **Batch Export**: Export animation frames at custom intervals
- **Multi-language UI**: Chinese and English interface support
- **Customizable Export**: Control texture size, interpolation algorithms, and more

### System Requirements

- **Godot Version**: 4.0 or higher (recommended)
- **Platform**: Windows, macOS, Linux
- **Graphics**: OpenGL 3.3 / Vulkan compatible GPU

### Installation

1. Copy the `addons/threeToTwo` folder to your Godot project's `addons` directory
2. Enable the plugin in Project Settings → Plugins
3. Click the "3D To 2D" button in the editor toolbar to open the plugin
